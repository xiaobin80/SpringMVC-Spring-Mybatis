Prepare:    
    1. Environment
    Set Maven(M2_home) in System variables.    
    1) Create an M2_HOME System Variable.    
    2) Add%M2_HOME%\bin; to your PATH variable.    
    
    2. Import Project    
    1) Open ME    
    2) Import->Maven->Existing Maven Projects    

Step 1: war       
    1) Export -> WAR file        
    2) Browse - > Save    
    3) Move war file to Tomcat webapps
